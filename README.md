# littletime bot

это telegram-бот с расширяемой функциональностью. в настоящее время проект находится в активной разработке.

на данный момент реализована только базовая функция:  
автоматическое приветствие новых участников при входе в группу. приветствие отправляется каждому пользователю, даже если он ранее уже состоял в чате.
Статистика опубликованных сообщений и сколько минут за всё время отправленно кружок/голосовое сообщение

в ближайшее время планируется добавление новых возможностей. следите за обновлениями!
